# Phase 1 Progress Memo for Overleaf

This folder is a self-contained Overleaf upload package for the one-page,
two-column Japanese phase-1 progress memo.

## Overleaf settings

- Main file: `main.tex`
- Compiler: `LuaLaTeX`

## Included figures

- `figures/fig_phase1_memo_scenario_mean_response_time.pdf`
- `figures/fig_phase1_memo_scenario_p95_response_time.pdf`
- `figures/fig_phase1_memo_b2_advantage_heatmap.pdf`

The memo intentionally describes the current work as a preliminary Monte Carlo
simulation, not as a complete discrete-event simulation or a real congestion
model.
